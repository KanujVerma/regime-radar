Market Stress Sandbox — Where does it live?

The sandbox lets users adjust key market inputs (VIX, drawdown, returns, etc.) and see model output change in real time. It uses the trained transition model — no new backend needed.

Expandable section on Current State

An st.expander("Scenario Explorer") at the bottom of the Current State page. Pre-populated with today's values so the user can immediately see how tweaks change today's risk reading.

Pros

  • Immediate context — "what if VIX was 30 today?"
  • No new nav

Cons

  • Current State page becomes complex
  • Threshold tuning has no natural home here

The sandbox controls

Regardless of placement, these sliders are pre-seeded from current live values and drive a live model score:

🔧 VIX level
🔧 VIX 5-day change
🔧 Realized vol percentile
🔧 Drawdown
🔧 20-day return
🔧 Distance from SMA-50

Output: baseline risk → adjusted risk → delta, plus updated narrative.
