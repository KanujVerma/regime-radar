What this scenario means
Raising VIX to 28.5 with rising fear (+4.2 over 5 days), elevated choppiness (72nd percentile), and a recent −4.2% return pushes transition risk from 18% to 47% — nearly tripling the baseline. The model is most sensitive to the VIX spike and fast-rising fear combination. At this reading, the probability of entering an Elevated regime within the next week roughly doubles.

How this works in React: Each slider change calls POST /scenario (debounced ~100ms) → backend loads xgb_transition in-process and returns baseline risk, scenario risk, regime probs, and top driver deltas → right panel updates via Framer Motion animated transitions. Presets snap all sliders at once and trigger a single API call. Threshold tuning reads from model metadata only — no API call.
