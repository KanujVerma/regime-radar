## Guiding principle

Regime Radar is a **credibility product**: its value is one trustworthy number (current regime + transition risk). Today that number is **self-undermining** — live transition scores reach ~76% but the reliability table's validated range (`max_evaluated_p`) caps at 0.30, so the UI disclaims the score as a "directional stress signal, not a calibrated probability." Models last trained 2026-05-23; `eval_history` holds a single snapshot.

Polish and distribution amplify whatever trust the core number has. So the core number comes first.
